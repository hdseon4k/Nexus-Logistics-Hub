# Nexus-Logistics-Hub
현장 데이터를 기반으로 화주사와 관리자가 사용하는 통합 대시보드

주요 기능	| 상세 내용	| 비고 (증거력)
실시간 진척도 |	입고 -> 팔레트화 -> 적입 단계별 퍼센트(%) 표시	| 작업 누락 방지
색상별 재고 관리 |	출고 대기장 내 국가별(색상별) 팔레트 수량 가시화	| 오출고 방지
디지털 증거함 |	입고 시 사진 + 중량 측정 사진 + 컨테이너 적입 사진 매칭	| 클레임 방어용 (AX 핵심)
ERP 동기화	| 화주사 ERP와 Nexus-Edge 간 실시간 API 연동	| 수기 입력 제로화
